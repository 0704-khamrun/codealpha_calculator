# 🧮 Calculator App

A simple, responsive, and user-friendly calculator built using modern frontend technologies. This application performs basic arithmetic operations with a clean and intuitive user interface.

## ✨ Features

* Basic arithmetic operations (+, -, ×, ÷)
* Clear (C) and Delete (⌫) functionality
* Responsive design for mobile and desktop
* Real-time calculation display
* Clean and modern user interface
* Keyboard input support (optional)

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript 

### Optional Technologies

* React.js
* Vite
* Tailwind CSS
* Bootstrap
